## Installation

### Setup .env file in folder backend

```bash
MONGODB_URI=...
ACCESS_TOKEN_SECRET_KEY=...
REFRESH_TOKEN_SECRET_KEY=...
BUILD_MODE=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

```

### Build the app with docker-compose

```bash
docker compose up -d
```
